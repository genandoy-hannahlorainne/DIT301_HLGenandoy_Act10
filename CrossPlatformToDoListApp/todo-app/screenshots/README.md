# Screenshots

After running the app, take screenshots and place them here:

1. **add_task.png** - Screenshot showing the text input and add button
2. **task_list.png** - Screenshot showing multiple tasks in the list
3. **delete_task.png** - Screenshot showing the delete button interaction

## How to Take Screenshots

### On Android Emulator
- Press Ctrl + S (Windows/Linux) or Cmd + S (Mac)
- Or use the camera icon in the emulator toolbar

### On iOS Simulator
- Press Cmd + S
- Screenshots are saved to Desktop

### On Physical Device
- Android: Power + Volume Down
- iOS: Side Button + Volume Up
